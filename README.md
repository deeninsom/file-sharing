## Installation

```bash
$ Clone repository to your directory
$ npm install
$ Setup your database on diretory config/config.js && config/sync-db.js
$ Run automatic create database with "npm run database"
$ Create folder uploads
```

## Running the app

```bash
# development
$ npm run start
```
